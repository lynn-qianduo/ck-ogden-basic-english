# Basic English 850词 · 智能学习平台

一个基于 C.K. Ogden Basic English 850 核心词汇的智能英语学习工具，纯前端实现，无需后端服务器。

![版本](https://img.shields.io/badge/version-1.0-blue)
![许可证](https://img.shields.io/badge/license-MIT-green)
![平台](https://img.shields.io/badge/platform-web-lightgrey)

---

## 📚 功能特点

| 功能 | 说明 |
|------|------|
| 📚 850 核心词汇 | 8 大分类，遵循 Basic English 规范 |
| ⭐ 收藏夹 | 收藏重点单词，方便复习 |
| ❌ 错词本 | 自动记录错误单词，针对性强化 |
| 📝 单词测试 | 英译中/中译英随机测试，10题/轮 |
| 📈 学习统计 | 掌握率、进度可视化，支持数据导入/导出 |
| 🤖 AI 造句 | 支持硅基流动/DeepSeek/通义千问，自动降级到本地例句库 |
| 📅 学习计划 | 基于艾宾浩斯遗忘曲线，三种学习模式 |
| 🔍 搜索高亮 | 实时搜索，自动滚动到匹配结果 |
| 🔊 发音功能 | 英音/美音发音（有道词典 API + 浏览器 TTS 降级） |
| 💾 数据备份 | 一键导出/导入学习数据，永不丢失 |
| 📱 移动端优化 | 触摸适配、响应式设计 |

## 🚀 快速开始

### 本地运行
1. 下载或克隆本项目
2. 确保 `data/words.csv` 文件存在
3. 用浏览器打开 `index.html`
4. 开始学习！

### 部署到云端
支持任意静态托管服务：
- **GitHub Pages**：将项目推送到 GitHub，开启 Pages 功能
- **Vercel**：`vercel --prod`
- **Netlify**：拖拽文件夹即可部署
- **CloudFlare Pages**：连接 GitHub 自动部署

## 🔧 AI 造句配置（可选）

1. 点击「AI 造句」进入模块
2. 选择提供商（硅基流动/DeepSeek/通义千问）
3. 点击「配置 API Key」输入你的密钥
4. 不配置则自动使用本地例句库（严格遵循 850 词规范）

> 💡 提示：AI 调用失败时会自动降级到本地例句库，保证功能可用。

---

## 💾 数据备份与恢复

1. 点击「学习统计」进入统计页面
2. 点击「导出数据」保存 JSON 备份文件
3. 需要恢复时点击「导入数据」选择备份文件

> 💡 提示：所有学习数据（收藏、错词、已学记录、学习计划、AI 历史）都会一起备份。
